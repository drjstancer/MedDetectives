# MedDetectives

MedDetectives is a facilitator-led, hybrid educational escape-room experience designed for undergraduate pre-med learners.

The physical room experience is the primary learning environment.

The digital platform serves as a lightweight hybrid coordination layer that supports:
- collaborative clinical reasoning,
- evidence synthesis,
- participant progression,
- facilitator orchestration,
- QR-based discoveries,
- and immersive educational storytelling.

The experience emphasizes:
- collaborative meaning-making,
- iterative reasoning,
- communication,
- uncertainty tolerance,
- and reflective synthesis.

Reasoning quality determines progression.
